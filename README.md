# 🌳 PRODIGY_DS_03 - Decision Tree Classifier

## 📌 Internship

This project was completed as part of the **Data Science Internship** at **Prodigy InfoTech**.

---

## 📖 Project Overview

The objective of this project is to build a **Decision Tree Classifier** to predict whether a customer will subscribe to a term deposit based on demographic and banking information.

---

## 🎯 Objective

- Perform Exploratory Data Analysis (EDA)
- Visualize important customer patterns
- Build a Decision Tree Classifier
- Evaluate model performance
- Identify important predictive features

---

**Bank Marketing Dataset**

## 🔑 Key Insights

- The dataset is imbalanced.
- Customer age mostly ranges between 30–50 years.
- Housing loans are common among customers.
- Duration is one of the most influential features.
- Decision Tree provides good classification performance.
